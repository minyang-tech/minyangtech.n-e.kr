# IXO Engine 개인정보 처리방침

시행일: 2026-05-30

민양테크는 IXO Engine 이용 과정에서 처리될 수 있는 정보의 범위와 이용자의 주의사항을 명확히 안내하기 위해 본 개인정보 처리방침을 공개합니다. IXO Engine은 기본적으로 로컬 우선 방식으로 동작하는 비주얼 앱 빌더이며, 프로젝트 데이터는 사용자의 기기에 저장됩니다.

## 1. 처리 목적

IXO Engine은 다음 목적을 위해 필요한 범위에서만 네트워크 기능을 사용합니다.

| 구분 | 목적 |
|---|---|
| 업데이트 확인 | 앱 시작 시 1회 최신 버전 여부 확인 및 사용자가 업데이트 적용을 선택한 경우 업데이트 흐름 진행 |
| 네트워크 계열 노드 | 사용자가 직접 구성한 HTTPS 요청 또는 외부 브라우저 열기 기능 실행 |
| 외부 링크 | 문서, GitHub Release, 개인정보 처리방침 등 사용자가 선택한 외부 페이지 열기 |

프로젝트 파일, 자동 저장 데이터, Canvas Builder 구성 정보는 민양테크 서버로 자동 업로드되지 않습니다.

## 2. 처리될 수 있는 정보

| 구분 | 처리될 수 있는 정보 |
|---|---|
| 업데이트 확인 | GitHub API 연결 과정에서 발생할 수 있는 IP 주소 및 통신 메타데이터 |
| 사용자가 만든 네트워크 노드 | 사용자가 입력한 URL, 요청 본문, 헤더, 토큰, 개인정보 등 |
| 브라우저 열기 노드 | 사용자가 지정한 외부 URL |
| 파일 감시 노드 | 사용자가 직접 선택하고 승인한 로컬 경로 정보 |

IXO Engine 자체는 사용자의 계정 정보, 프로젝트 내용, 입력값을 민양테크 서버로 자동 수집하지 않습니다. 다만 사용자가 직접 외부 서버로 전송하도록 구성한 데이터는 해당 외부 서비스 운영자에게 전달될 수 있습니다.

## 3. 네트워크 노드 동의

V1.1.3부터 네트워크 계열 노드를 처음 사용할 때 IXO Engine 내부 UI로 동의서가 표시됩니다.

- 동의하지 않아도 앱 사용은 계속할 수 있습니다.
- 동의하지 않으면 HTTPS 요청, 브라우저 열기 등 네트워크 계열 노드가 제한됩니다.
- 설정의 네트워크 항목에서 동의서를 다시 볼 수 있습니다.
- 설정에서 동의를 취소할 수 있습니다.

## 4. 보안 조치

IXO Engine은 네트워크 계열 기능에 다음 보호 조치를 적용합니다.

- HTTPS 강제
- URL 파싱 실패 차단
- localhost, loopback, 사설망 주소 차단
- 외부 요청 실행 전 사용자 승인
- 요청 타임아웃 적용
- in-flight 중복 요청 방지
- 로그 URL 마스킹
- 브라우저 열기와 외부 링크에도 동일 정책 적용
- 파일 감시 노드에 별도 사용자 승인 적용
- 스크립트 노드 제한 실행 모드와 전체 JavaScript 모드 경고 UI 적용

## 5. 보유 및 이용 기간

- 민양테크는 업데이트 확인과 관련해 별도 서버에 개인정보를 저장하지 않습니다.
- 로컬 프로젝트와 자동 저장 데이터는 사용자가 삭제하거나 초기화할 때까지 사용자 기기에 남을 수 있습니다.
- 사용자가 외부 서버로 전송한 정보의 보유 기간은 해당 외부 서비스의 정책을 따릅니다.

## 6. 제3자 제공 및 국외 이전

- 민양테크는 IXO Engine을 통해 수집한 개인정보를 별도 제3자에게 제공하지 않습니다.
- 업데이트 확인에는 GitHub API가 사용될 수 있습니다.
- 사용자가 직접 구성한 네트워크 노드가 호출하는 외부 서비스는 민양테크가 운영하지 않습니다.
- 외부 서비스가 국외에 있을 수 있으므로, 사용자는 해당 서비스의 개인정보 처리방침과 보안 수준을 별도로 확인해야 합니다.

## 7. 이용자의 권리와 행사 방법

- 로컬 프로젝트와 자동 저장 데이터는 앱의 초기화 또는 삭제 기능으로 삭제할 수 있습니다.
- 네트워크 노드 동의는 설정에서 취소할 수 있습니다.
- 외부 서비스로 전송한 정보의 열람, 정정, 삭제, 처리정지는 해당 외부 서비스 운영자에게 요청해야 합니다.

## 8. 이용자 주의사항

- 비밀번호, 인증 토큰, 주민등록번호, 결제 정보 등 민감한 정보를 신뢰할 수 없는 서버 또는 스크립트에 입력하지 마십시오.
- 사용자가 직접 입력하거나 연결한 URL, 서버, 데이터에 대해서는 사용자가 직접 확인해야 합니다.
- 외부 서비스의 내용, 보안, 개인정보 처리 방식은 민양테크가 통제하지 않습니다.
