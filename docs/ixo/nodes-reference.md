# IXO Engine 노드 상세 문서

이 문서는 `node-library.md`와 같은 V1.1.3 기준을 사용합니다. 자세한 최신 노드 목록은 Node Library 페이지를 기준으로 확인하십시오.

## V1.1.3에서 특히 확인할 항목

| 항목 | 설명 |
|---|---|
| 입력 칸 | 사용자 입력이 UI와 런타임 컨텍스트에 정상 반영됩니다. |
| 노드 검색 | `Ctrl+F`와 사이드바 검색창 모두 노드 이름, 번역명, 타입, 그룹을 기준으로 검색합니다. |
| 전역변수/지역변수 | 사이드바 변수 영역에서 만들고 템플릿 문법으로 참조합니다. |
| 함수 편집 | 함수 편집기는 브라우저형 탭으로 열립니다. |
| Canvas Builder | UI 선택, 수정, 반응형 미리보기 가로/세로 조절을 지원합니다. |
| 네트워크 노드 | 동의서, HTTPS 강제, 사설망 차단, 타임아웃, 로그 마스킹 정책을 따릅니다. |

자세한 내용은 `/docs/ixo/nodes.html`을 확인하십시오.
