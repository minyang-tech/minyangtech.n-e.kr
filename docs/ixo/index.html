IXO Engine

Visual Node-Based Game Engine

소개

IXO Engine은 복잡한 텍스트 코딩 없이 노드 연결만으로 로직과 인터랙션을 설계할 수 있는 비주얼 프로그래밍 기반의 앱 제작 엔진입니다.

  • 엔진 명칭 IXO Engine (v1.0)
  • 주요 기능 노드 에디터, 실시간 런타임 프리뷰, 변수 시스템
  • 개발 언어 React, JavaScript (Web-based)
  • 지원 플랫폼 Web Browsers, Desktop (via Electron)
현재 안정화 및 기능 확장 단계에 있습니다.

핵심 특징

  • Intuitive Editor 드래그 앤 드롭 방식으로 직관적인 로직 설계가 가능합니다.
  • Template Engine {{variable}} 문법을 통해 데이터 치환과 조건 평가를 지원합니다.
  • Extensibility Script 노드를 사용하여 자신만의 커스텀 JS 로직을 추가할 수 있습니다.

내보내기 방법

  • IXO Engine에서 앱을 다 제작 후, .exe (리눅스 유저의 경우 다르게 나옵니다)로 내보내는법.
  • 1. 진행중인 파일을 저장합니다.
  • 2. 화면 좌측, [File]을 클릭합니다.
  • 3. Export 버튼을 클릭해주세요.
  • 4. 앱 아이콘과 앱 이름을 지정해주세요.
  • 5. 경로를 지정해주세요, 하지않을시 내보내기가 되지않습니다.
  • 6. 확장자를 지정해주세요. 가능한 확장자는 아래와 같습니다.
    • .exe (PC, Windows.)
    -> 즉시 실행형, 설치형
    .msi (PC)
    .app (PC, Mac)
    .apk (Mobile, Android etc.)
    .ipa(Mobile, iPhone / Ipad etc.)
  • 7. 내보내기 버튼을 눌러주세요.
