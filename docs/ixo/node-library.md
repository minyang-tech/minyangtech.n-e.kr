# IXO Engine Node Library

이 문서는 IXO Engine V1.1.3 기준으로 제공되는 노드와 관련 편집 기능을 정리합니다.

## 상태 표기

| 상태 | 의미 |
|---|---|
| Stable | 현재 공개 빌드에서 실제로 사용할 수 있는 기능입니다. |
| Protected | 사용할 수 있지만 보안 동의, 권한 확인 또는 제한 실행 정책이 적용됩니다. |
| Editor | 노드 자체가 아니라 편집기 기능으로 제공되는 항목입니다. |

## 공통 필드

| 필드 | 설명 |
|---|---|
| Label | 노드 또는 UI 요소의 표시 이름입니다. |
| Value | 노드가 계산하거나 전달하는 기본 값입니다. |
| Ref Key | 다른 노드나 UI가 `{{refKey}}`로 참조할 수 있는 이름입니다. |
| Group Label | 여러 노드를 시각적으로 묶기 위한 보조 라벨입니다. |
| Scene | 노드와 UI가 속한 화면 단위입니다. |

## Canvas Builder 연동

- Canvas Builder에서 Text, Image, Button, Custom Button, Vector, Input, Container를 추가하면 대응되는 Visual 노드가 자동으로 생성되거나 연결됩니다.
- Inspector에서 UI 값을 수정하면 연결된 노드의 `Value` 또는 `Ref Key`가 함께 갱신됩니다.
- UI를 삭제하면 의미 없는 연결 노드가 남지 않도록 정리됩니다.
- 이미지 asset은 Canvas Builder로 드래그해 이미지 UI와 이미지 노드를 만들 수 있습니다.
- `.ixo` asset은 Node Workspace 또는 Canvas Builder로 드래그해 포함된 노드와 UI를 현재 프로젝트에 붙여넣을 수 있습니다.
- `P` 키를 누르면 선택한 UI와 연결된 노드 위치로 이동할 수 있습니다.
- 반응형 미리보기는 Desktop, Tablet, Mobile 모드를 제공하며, V1.1.3부터 가로와 세로 크기 조절을 지원합니다.

### 네트워크 보안 정책

- 네트워크 계열 노드는 첫 사용 전에 동의서를 표시합니다.
- 동의서는 설정의 네트워크 항목에서 다시 볼 수 있고 취소할 수 있습니다.
- HTTPS만 허용합니다.
- URL 파싱 실패, localhost, loopback, 사설망 주소는 차단합니다.
- 요청에는 타임아웃과 in-flight 중복 방지가 적용됩니다.
- 로그에는 URL 민감 정보가 마스킹됩니다.
- 브라우저 열기와 외부 링크도 같은 정책을 따릅니다.
