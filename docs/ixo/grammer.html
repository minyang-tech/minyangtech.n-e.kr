Template Grammar

변수 참조 및 조건식 사용법

기본 문법

IXO Engine의 Inspector 창에서 Value 또는 조건식 입력란에 변수를 참조할 때 사용합니다.

{{variableName}}

실전 활용 예시

  1. 변수 생성: Input Field 노드의 Ref Key를 playerName으로 설정합니다.
  2. 변수 출력: Add Text 노드의 Value에 안녕하세요, {{playerName}}님! 이라고 입력합니다.
  3. 결과: 사용자가 입력한 이름이 텍스트에 실시간으로 적용되어 출력됩니다.

조건식 체인 (Logical Logic)

If / Else 노드 등에서 여러 조건을 복합적으로 검사할 때 사용합니다. AND와 OR 연산자를 지원합니다.

{{score}} > 10 AND {{role}} == 'admin' OR {{username}} == 'root'

평가 순서: 왼쪽에서 오른쪽으로 AND를 우선 평가한 후 OR 체인을 처리합니다.
