# IXO Engine

IXO Engine ist ein visueller Node-App-Builder zum Erstellen von Desktop-Apps mit Nodes, Szenen, Assets, Actions und UI-Panels.

## Funktionen

- Abläufe mit visuellen Nodes bauen.
- UI mit Canvas Builder und UI Viewer entwerfen.
- Szenen, Variablen, Assets und Actions für praktische Apps nutzen.
- Windows- und Linux-Builds mit saubererer Runtime-Oberfläche exportieren.

## Sicherheitsmodell

Netzwerk-Nodes benötigen die Zustimmung des Erstellers. HTTPS wird erzwungen, localhost/private Netzwerke werden blockiert, URLs in Logs maskiert und riskante Actions laufen durch dieselbe Berechtigungslogik.

## Plattformunterstützung

App-Nutzung und Export werden primär auf Windows und Linux geprüft. macOS-Desktop-App-Support wird derzeit nicht angeboten.
