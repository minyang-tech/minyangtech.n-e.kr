# 편집 지침
1. 해당 저장소에서 ../docs 파일 내부만 편집 가능합니다.
2. news-data.json의 편집을 일부분 허용합니다.  
2-1. news-data.json에서, 신규 뉴스 작성은 허용되지않습니다. 허용범위는 "오탈자의 수정"입니다.
3. 해당저장소에 악성코드, 스파게티코드, 멀웨어 등 허가되지않은 코드를 심는것은 허용되지않습니다.
4. 대한민국 법률에 위법되는 행위는 허용되지않습니다.  
4-1. 기여자가 거주하고있는 지역에 법률 및 대한민국 법률을 포함합니다.  
5. 특정 인물에 대한 비방을 어떠한 방식을 통해 작성하는행위는 허용되지않습니다.
