# IXO Engine

IXO Engine は、ノード、Scene、Asset、Action、UI パネルを使ってデスクトップアプリを作成するビジュアルノード型アプリビルダーです。

## できること

- ビジュアルノードで処理フローを作成します。
- Canvas Builder と UI Viewer で UI を設計します。
- Scene、変数、Asset、Action を使って実用的なアプリを作れます。
- Windows と Linux 向けに、編集画面の痕跡を減らしたランタイムを export できます。

## 安全モデル

ネットワーク系ノードは制作者の同意が必要です。HTTPS を強制し、localhost / プライベートネットワークをブロックし、ログでは URL をマスクします。危険な Action も同じ承認ポリシーを通ります。

## 対応プラットフォーム

アプリ利用と export 検証は Windows / Linux 中心です。macOS デスクトップアプリは現在サポートしていません。
