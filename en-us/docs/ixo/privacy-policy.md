# Network Node Privacy Policy

IXO Engine provides network-family nodes such as HTTPS Request and Browser Open. These nodes are disabled until the project creator consents.

## What can be processed

A project may contain URLs, request settings, labels, and node metadata entered by the creator. MinyangTech does not operate the external servers contacted by a user's project.

## User responsibility

Do not send passwords, tokens, personal information, or sensitive data to untrusted endpoints. Check the destination before running or exporting network-enabled projects.

## Safety controls

IXO Engine enforces HTTPS, blocks localhost/private network targets, applies timeouts, masks URLs in logs, and records a non-visible safety signal in exported builds when network consent was granted.
