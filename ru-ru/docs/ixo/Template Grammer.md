# Template Grammar

Шаблоны IXO вставляют значения предыдущих узлов в текст.

## Базовая форма

Используйте двойные фигурные скобки:

`Hello, {{username}}`

Если в context есть `username = "Alex"`, результат будет `Hello, Alex`.

## Условия

Поддерживаются `==`, `!=`, `>`, `<`, `>=`, `<=`, `AND`, `OR`.

Пример:

`{{score}} >= 60 AND {{name}} != ""`

## Безопасность

Значения шаблонов считаются данными. Не вставляйте секреты, токены или пароли в недоверенные скрипты или сетевые действия.
