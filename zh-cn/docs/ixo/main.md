# IXO Engine

IXO Engine 是一款可视化节点式应用构建器，可用节点、场景、资源、动作和 UI 面板创建桌面应用。

## 功能

- 用可视化节点构建流程。
- 通过 Canvas Builder 和 UI Viewer 设计界面。
- 使用场景、变量、资源和动作制作实用应用。
- 导出 Windows 与 Linux 构建，并尽量移除编辑器痕迹。

## 安全模型

网络类节点需要创作者同意。系统强制 HTTPS，阻止 localhost / 私有网络访问，在日志中遮蔽 URL，危险动作也会经过同一审批策略。

## 平台支持

应用使用与导出验证以 Windows / Linux 为主。目前不提供 macOS 桌面应用支持。
